Esercizi aggiuntivi

Individuare il dominio delle seguenti funzioni e rappresentarlo sul piano cartesiano:

  1. \[ f\left(x\right) = \dfrac{6x -5}{5x^2 -3x -2} \]

    Soluzione
    \[ x \neq -\dfrac{2}{5} \quad\text{oppure}\quad x \neq 1 \]


  2. \[ f\left(x\right) = \sqrt{-6x -10} \]

    Soluzione
    \[ x \leq -\dfrac{5}{3} \]


  3. \[ f\left(x\right) = ln\left(18x^2 - 8\right) + \sqrt{2x - 10} \]

    Soluzione
    \[ x \geq 5 \]


  4. \[ f\left(x\right) = \sqrt{\dfrac{1 - x}{2x^2 +x +1}} \]

    Soluzione
    \[ x \lt -1 \quad\text{oppure}\quad \dfrac{1}{2} \lt x \leq 1 \]


  5. \[ f\left(x\right) = \dfrac{e^{\frac{1}{x +1}}}{\sqrt{9x^2 -1}} \]

    Soluzione
    \[ x \lt -1 \quad\text{oppure}\quad -1 \lt x \lt -\dfrac{1}{3} \quad\text{oppure}\quad x \gt \dfrac{1}{3} \]
