Esercizio 1

Assegnata una funzione \(f\), sappiamo che


Dominio:
\[ -6 \lt x \lt -2 \quad\lor\quad x \gt 0 \]

Segno:
\[ f\left(x\right) \gt 0 \quad\text{se}\quad -6 \lt x \lt -4 \quad\lor\ x\gt2 \]


Rappresentare dominio e segno nel grafico cartesiano.



Esercizio 2

Data la funzione \[ f(x) = \sqrt{-6x^2 -x +5} \]


  • stabilire il suo dominio;
  • studiare come varia il suo segno al variare dell'input \(x\);
  • determinare se è pari, dispari o nessuna delle due.
  • Graficare le informazioni ottenute.


Esercizio 3

Data la funzione \[ f(x) = \dfrac{9-x^2}{x^2 - 1} \]


  • stabilire il suo dominio;
  • studiare come varia il suo segno al variare dell'input \(x\);
  • Graficare le informazioni ottenute.
  • determinare se è pari, dispari o nessuna delle due.


Esercizio extra

Scrivere una funzione che abbia


  • Dominio

    \[ x \neq 3 \]


  • Segno

    \( f\left(x\right) \gt 0 \quad\text{se}\quad x \lt 1 \quad\lor\quad x \gt 3 \)

    \( f\left(x\right) = 0 \quad\text{se}\quad x = 1 \)

    \( f\left(x\right) \lt 0 \quad\text{se}\quad 1 \lt x \lt 3 \)


(L'esercizio extra è veramente extra. Già facendo i 3 esercizi avete la possibilità di accedere ad un signor voto)
