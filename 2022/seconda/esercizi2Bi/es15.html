Esercizio 3

Studiare il segno del seguente polinomio al variare del valore della \(x\): \[ -9x^2 +11x -2 \]


    Soluzione

  • \(-9x^2 +11x -2 \gt 0 \quad\text{se}\quad \dfrac{2}{9} \lt x \lt 1\)


  • \(-9x^2 +11x -2 = 0 \quad\text{se}\quad x = \dfrac{2}{9} \quad\text{oppure}\quad x = 1\)


  • \(-9x^2 +11x -2 \lt 0 \quad\text{se}\quad x \lt \dfrac{2}{9} \quad\text{oppure}\quad x \gt 1\)

Soluzione

\[ x = -\dfrac{2}{9} \quad\text{oppure}\quad x = -\dfrac{2}{5} \quad\text{oppure}\quad x = \dfrac{1 + \sqrt{6}}{5} \quad\text{oppure}\quad x = \dfrac{1 - \sqrt{6}}{5} \]
