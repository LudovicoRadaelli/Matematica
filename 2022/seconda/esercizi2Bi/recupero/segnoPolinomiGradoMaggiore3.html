Esercizi aggiuntivi

  1. \[ f\left(x\right) = 4x^{4} - 9 \]

  2. \[ f\left(x\right) = 75x^3 -27x \]

  3. \[ f\left(x\right) = 4x^3 -16x^2 +9x -36 \]

Soluzione


    • \(\boldsymbol{f\left(x\right)} \gt 0\) se \[x \lt -\sqrt{\dfrac{3}{2}} \quad\text{oppure}\quad x \gt \sqrt{\dfrac{3}{2}}\]

    • \(\boldsymbol{f\left(x\right)} \lt 0\) se \[-\sqrt{\dfrac{3}{2}} \lt x \lt \sqrt{\dfrac{3}{2}}\]

    • \(\boldsymbol{f\left(x\right)} = 0\) se \[x = -\sqrt{\dfrac{3}{2}} \quad\text{oppure}\quad x = \sqrt{\dfrac{3}{2}}\]

    • \(\boldsymbol{f\left(x\right)} \gt 0\) se \[-\dfrac{3}{5} \lt x \lt 0 \quad\text{oppure}\quad x \gt \dfrac{3}{5}\]

    • \(\boldsymbol{f\left(x\right)} \lt 0\) se \[x = -\dfrac{3}{5} \quad\text{oppure}\quad x = 0 \quad\text{oppure}\quad x = \dfrac{3}{5}\]

    • \(\boldsymbol{f\left(x\right)} = 0\) se \[x \lt -\dfrac{3}{5} \quad\text{oppure}\quad 0 \lt x \lt \dfrac{3}{5}\]

    • \(\boldsymbol{f\left(x\right)} \gt 0\) se \[ x \gt 4 \]

    • \(\boldsymbol{f\left(x\right)} \lt 0\) se \[ x = 4 \]

    • \(\boldsymbol{f\left(x\right)} = 0\) se \[ x \lt 4 \]
