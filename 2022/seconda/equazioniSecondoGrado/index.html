Data l'equazione \[ \color{red}{a}\,\color{black}{x^2}\, + \color{blue}{b}\,\color{black}{x}\, + \color{darkorange}{c}\,\, \color{black}{}= 0 \] la forumula risolutiva è \[ x = \dfrac{-\color{blue}{b}\color{black}{} \pm \sqrt{\color{blue}{b}\color{black}{}^2 - 4 \cdot \color{red}{a}\color{black} \cdot \color{darkorange}{c}\color{black}}}{2\color{red}{a}} \]



2) Numero di soluzioni



Data l'equazione \[ \color{red}{a}\,\color{black}{x^2}\, + \color{blue}{b}\,\color{black}{x}\, + \color{darkorange}{c}\,\, \color{black}{}= 0 \] sappiamo che la formula risolutiva è \[ x = \dfrac{-\color{blue}{b}\color{black}{} \pm \sqrt{\color{blue}{b}\color{black}{}^2 - 4 \cdot \color{red}{a}\color{black} \cdot \color{darkorange}{c}\color{black}}}{2\color{red}{a}} \] Indichiamo con la lettera \(\Delta\) il contenuto della radice presente nella formula, ovvero \[\Delta = \color{blue}{b}^2 \color{black}{-4}\,\color{red}{a}\,\color{darkorange}{c} \] il numero di soluzioni dipende dal segno di \(\Delta\):


  • Se \(\Delta \gt 0\),
    allora l'equazione ha due soluzioni;

  • Se \(\Delta = 0\),
    allora l'equazione ha una soluzioni;

  • Se \(\Delta \lt 0\),
    allora l'equazione ha zero soluzioni;

3) Casi particolari

Ci sono alcuni casi in cui possiamo usare strade alternative e più rapide rispetto all'applicaizione della formula.


3.1) \(\,\,\)Se \(\boldsymbol{b = 0}\)


Nel caso in cui \(b\) sia \(0\), il termine con la \(x\) non compare nell'equazione: \[ ax^2 + c = 0 \]

Esaminiamo un caso concreto: \[ 2x^2 - 18 = 0 \] In questo caso possiamo isolare la \(x^2\).

Primo passo, spostiamo il \(-18\) dall'altra parte dell'uguale. \[ 2x^2 = 18 \] Dividiamo per \(2\)

\[ x^2 = \dfrac{18}{2} \] ovvero \[ x^2 = 9 \]

Bene, sappiamo che \(x\) è quel numero che elevato al quadrato fa \(9\)... la radice quadrata di \(9\)! \[ x = \sqrt{9} \]

Ma vediamo che \(\sqrt{9}\) non è l'unico numero che elevato al quadrato fa \(9\), anche \(\left(-\sqrt{9}\right)^2\) fa \(9\).

Quindi \[ x = \sqrt{9} \quad\text{oppure}\quad x = -\sqrt{9} \]

Esempi

Risolviamo l'equazione \[ 3x^2 - 8 = 0 \] Isoliamo la \(x^2\): \[ 3x^2 = 8 \] \[ x^2 = \dfrac{8}{3} \] Dunque \[ x = \sqrt{\dfrac{8}{3}} \quad\text{oppure}\quad x = -\sqrt{\dfrac{8}{3}} \]




3.2) \(\,\,\) Se \(\boldsymbol{c = 0}\)


Nel caso in cui \(c\) sia \(0\), il termine senza \(x\) non compare nell'equazione: \[ ax^2 + bx = 0 \]

Esaminiamo un caso concreto: \[ 5x^2 +3x = 0 \] Notate che il polinomio \(5x^2 +3x\) può essere riscritto come \(x\cdot\left(5x + 3\right)\).
L'equazione quindi si riscrive così: \[ x\cdot\left(5x + 3\right) = 0 \] Leggiamolo in questo modo:

\[ \underset{\color{red}{\text{il numero \(x\)}}}{\underbrace{\color{red}{x}}} \underset{\color{darkgreen}{\,\,\text{moltiplicato}}}{\color{darkgreen}{\underbrace{\cdot}}} \underset{\,\,\color{blue}{\text{per il numero \(5x +3\)}}}{\underbrace{\color{blue}{\left(5x + 3\right) }}} \underset{\text{fa \(0\)}}{\underbrace{= 0}} \]


Ma allora deve esser vero che almeno uno tra \(\color{red}{x}\) e \(\,\color{blue}{5x +3}\,\) è \(0\)
(sennò il loro prodotto non potrebbe essere \(0\)!).
Quindi

\[ \color{red}{x\,} \color{black}{=0} \quad\text{oppure}\quad\color{blue}{5x +3\,}\color{black}{ = 0} \]

\[ x = 0 \quad\text{oppure}\quad 5x = -3 \]

\[ x = 0 \quad\text{oppure}\quad x = -\dfrac{3}{5} \]

Esempi

Risolviamo l'equazione \[ 2x^2 - 7x = 0 \] Mettiamo in evidenza la \(x\) al primo membro: \[ x \cdot \left(2x - 7\right) = 0 \] Dunque \[ x = 0 \quad\text{oppure}\quad 2x - 7 = 0 \] Svolgiamo l'equazione di primo grado a destra: \[ x = 0 \quad\text{oppure}\quad 2x = 7 \] \[ x = 0 \quad\text{oppure}\quad x = \dfrac{7}{2} \]
