Esercizio 1

Studiare il segno della seguente espressione al variare del valore di \(x\): \[ \dfrac{5x -3}{3x^2 -6x + 1} \]

Soluzione

\(E\left(x\right) = \dfrac{5x -3}{3x^2 -6x + 1}\)


\(\boldsymbol{E\left(x\right) \gt 0}\) se


\( \dfrac{3 -\sqrt{6}}{3} \lt x \lt \dfrac{3}{5} \quad\text{oppure}\quad x \gt \dfrac{3 + \sqrt{6}}{3}\)



\(\boldsymbol{E\left(x\right) \lt 0}\) se


\( x \lt \dfrac{3 -\sqrt{6}}{3} \quad\text{oppure}\quad \dfrac{3}{5} \lt x \lt \dfrac{3 + \sqrt{6}}{3}\)



\(\boldsymbol{E\left(x\right) = 0}\) se


\( x = \dfrac{3}{5} \)



\(\boldsymbol{E\left(x\right) \quad \text{non è definita}\,\,\,}\) se


\( x = \dfrac{3 -\sqrt{6}}{3} \quad\text{oppure}\quad x = \dfrac{3 + \sqrt{6}}{3}\)



Esercizio 2

Studiare il segno del seguente polinomio al variare del valore di \(x\): \[ 18x^7 -21x^6 +5x^5 \]


Suggerimento
Prima scomponete il polinomio, successivamente studiate il segno dei fattori e realizzate il grafico dei segni.


Soluzione

\(E\left(x\right) = 18x^7 -21x^6 +5x^5\)


\(\boldsymbol{E\left(x\right) \gt 0}\) se


\(0 \lt x \lt \dfrac{1}{3} \quad\text{oppure}\quad x \gt \dfrac{5}{6}\)



\(\boldsymbol{E\left(x\right) \lt 0}\) se


\( x \lt 0 \quad\text{oppure}\quad \dfrac{1}{3} \lt x \lt \dfrac{5}{6}\)



\(\boldsymbol{E\left(x\right) = 0}\) se


\( x = 0 \quad\text{oppure}\quad x = \dfrac{1}{3} \quad\text{oppure}\quad x = \dfrac{5}{6}\)



Esercizio 3

\[ \left(15x^2 +7x -2\right)^3 \cdot \left(-5x +2\right)^2 \]


Soluzione

\(E\left(x\right) = \left(15x^2 +7x -2\right)^3 \cdot \left(-5x +2\right)^2\)


\(\boldsymbol{E\left(x\right) \gt 0}\) se


\(x \lt -\dfrac{2}{3} \quad\text{oppure}\quad \dfrac{1}{5} \lt x \lt \dfrac{2}{5} \quad\text{oppure}\quad x \gt \dfrac{2}{5}\)



\(\boldsymbol{E\left(x\right) \lt 0}\) se


\(-\dfrac{2}{3} \lt x \lt \dfrac{1}{5}\)



\(\boldsymbol{E\left(x\right) = 0}\) se


\( x = -\dfrac{2}{3} \quad\text{oppure}\quad x = \dfrac{1}{5} \quad\text{oppure}\quad x = \dfrac{2}{5}\)



Esercizio 4

Studiare il segno della seguente espressione al variare del valore di \(x\): \[ \dfrac{\left(5x -2\right) \cdot \left(-4x +5\right)}{6x^2 -17x +5} \]

Soluzione

\(E\left(x\right) = \dfrac{\left(5x -2\right) \cdot \left(-4x +5\right)}{6x^2 -17x +5}\)


\(\boldsymbol{E\left(x\right) \gt 0}\) se


\( x \lt \dfrac{1}{3} \quad\text{oppure}\quad \dfrac{2}{5} \lt x \lt \dfrac{5}{4} \quad\text{oppure}\quad x \gt \dfrac{5}{2}\)



\(\boldsymbol{E\left(x\right) \lt 0}\) se


\( \dfrac{1}{3} \lt x \lt \dfrac{2}{5} \quad\text{oppure}\quad \dfrac{5}{4} \lt x \lt \dfrac{5}{2}\)



\(\boldsymbol{E\left(x\right) = 0}\) se


\(x = \dfrac{2}{5} \quad\text{oppure}\quad x = \dfrac{5}{4} \)



\(\boldsymbol{E\left(x\right) \quad \text{non è definita}\,\,\,}\) se


\( x = \dfrac{1}{3} \quad\text{oppure}\quad x = \dfrac{5}{2} \)
