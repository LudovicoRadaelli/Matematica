Studio del segno del segno funzioni del tipo \(f\left(x\right)= \sqrt[n]{A\left(x\right)}\)

Dispense

Paragrafo 4) del materiale disponibile QUI



Esercizi

Studiare il segno delle seguenti funzioni al variare del valore della \(x\):


  1. \[ f\left(x\right) = \sqrt[5]{6x + 9} \]

  2. \[ f\left(x\right) = \sqrt{2x^2 -5x + 1} \]

  3. \[ f\left(x\right) = \sqrt[3]{-7x^2 -1} \]

Soluzione


    • \(f\left(x\right) \gt 0 \quad\) se \[ x \gt -\dfrac{3}{2} \]

    • \(f\left(x\right) = 0 \quad\) se \[ x = -\dfrac{3}{2} \]

    • \(f\left(x\right) \lt 0 \quad\) se \[ x \lt -\dfrac{3}{2} \]

    • \(f\left(x\right) \gt 0 \quad\) se \[ x \lt \dfrac{5 - \sqrt{17}}{4} \quad\text{oppure}\quad x \gt \dfrac{5 +\sqrt{17}}{4} \]

    • \(f\left(x\right) = 0 \quad\) se \[ x = \dfrac{5 - \sqrt{17}}{4} \quad\text{oppure}\quad x = \dfrac{5 +\sqrt{17}}{4} \]

    • \(f\left(x\right)\,\,\) non definita se \[ \dfrac{5 - \sqrt{17}}{4} \lt x \lt \dfrac{5 +\sqrt{17}}{4} \]

    3. \(f\left(x\right) \lt 0 \quad\) per ogni \(x \in \mathbb{R}\)
