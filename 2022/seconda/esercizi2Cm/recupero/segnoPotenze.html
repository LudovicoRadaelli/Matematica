Studio del segno del segno funzioni del tipo \(f\left(x\right)= A\left(x\right)^n\)

Riferimenti al libro di testo

Esercizi: pag. 830 dal 271 al 286.



Dispense

Paragrafo 3) del materiale disponibile QUI



Esercizi aggiuntivi

Studiare il segno delle seguenti funzioni al variare del valore della \(x\):


  1. \[ f\left(x\right) = \left(6x -5\right)^4 \]

  2. \[ f\left(x\right) = \left(5x -2\right)^5 \]

  3. \[ f\left(x\right) = \left(-4x -9\right)^{19} \]

Soluzione


    • \(f\left(x\right) \gt 0 \quad\) se \[ x \lt \dfrac{5}{6} \quad\text{oppure}\quad x \gt \dfrac{5}{6} \]

    • \(f\left(x\right) = 0 \quad\) se \[ x = \dfrac{5}{6} \]

    • \(f\left(x\right) \gt 0 \quad\) se \[ x \gt \dfrac{2}{5} \]

    • \(f\left(x\right) = 0 \quad\) se \[ x = \dfrac{2}{5} \]

    • \(f\left(x\right) \lt 0 \quad\) se \[ x \lt \dfrac{2}{5} \]

    • \(f\left(x\right) \gt 0 \quad\) se \[ x \lt -\dfrac{9}{4} \]

    • \(f\left(x\right) = 0 \quad\) se \[ x = -\dfrac{9}{4} \]

    • \(f\left(x\right) \lt 0 \quad\) se \[ x \gt -\dfrac{9}{4} \]
