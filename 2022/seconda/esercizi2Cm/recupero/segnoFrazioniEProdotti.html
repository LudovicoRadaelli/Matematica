Esercizi aggiuntivi

Studiare il segno delle seguenti funzioni al variare del valore della \(x\):


  1. \[ f\left(x\right) = \dfrac{5x -3}{-24x^2 +10x +6} \]

  2. \[ f\left(x\right) = \left(-3x +1\right)\cdot\left(x^2 -9\right) \]

  3. \[ f\left(x\right) = \dfrac{\left(1 -x\right)\cdot\left(7x^2 -5\right)}{8x +4} \]

Soluzione


    • \(f\left(x\right) \gt 0 \quad\) se \[ x \lt -\dfrac{1}{3} \quad\text{oppure}\quad \dfrac{3}{5} \lt x \lt \dfrac{3}{4} \]

    • \(f\left(x\right) = 0 \quad\) se \[ x= \frac{3}{5} \]

    • \(f\left(x\right) \lt 0 \quad\) se \[ -\dfrac{1}{3} \lt x \lt \dfrac{3}{5} \quad\text{oppure}\quad x \gt \dfrac{3}{4} \]

    • \(f\left(x\right)\,\,\) non è definita se \[ x = -\dfrac{1}{3} \quad\text{oppure}\quad x = \dfrac{3}{4} \]

    • \(f\left(x\right) \gt 0 \quad\) se \[ x \lt -3 \quad\text{oppure}\quad \dfrac{1}{3} \lt x \lt 3 \]

    • \(f\left(x\right) = 0 \quad\) se \[ x = -3 \quad\text{oppure}\quad x = \dfrac{1}{3} \quad\text{oppure}\quad x = 3 \]

    • \(f\left(x\right) \lt 0 \quad\) se \[ -3 \lt x \lt \dfrac{1}{3} \quad\text{oppure}\quad x \gt 3 \]

    • \(f\left(x\right) \gt 0 \quad\) se \[ -\sqrt{\dfrac{5}{7}} \lt x \lt -\dfrac{1}{2} \quad\text{oppure}\quad \sqrt{\dfrac{5}{7}} \lt x \lt 1 \]

    • \(f\left(x\right) = 0 \quad\) se \[ x = \pm\sqrt{\dfrac{5}{7}} \quad\text{oppure}\quad x = 1 \]

    • \(f\left(x\right) \lt 0 \quad\) se \[ x \lt -\sqrt{\dfrac{5}{7}} \quad\text{oppure}\quad -\dfrac{1}{2} \lt x \lt \sqrt{\dfrac{5}{7}} \quad\text{oppure}\quad x \gt 1 \]

    • \(f\left(x\right)\,\,\) non è definita se \[ x = -\dfrac{1}{2} \]
