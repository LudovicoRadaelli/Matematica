Segno di un polinomio di secondo grado

Riferimenti al libro di testo

Teoria: da pag. 799 a pag. 803;

Esercizi: pag. 817 dal 49 al 153.



Dispense aggiuntive

Paragrafi 8) delle disponibili QUA



Esercizi aggiuntivi

Studiare il segno dei seguenti polinomi di secondo grado:


  1. \[ -x^2 +5x +6 \]

  2. \[ 7x^2 +5 \]

  3. \[ 6x^2 - x +3 \]

  4. \[ -9x^2 +11x -2 \]

  5. \[ -4x^2 + 12x -9 \]

Soluzione


    • \( -x^2 +5x +6 \gt 0 \quad\text{se}\quad -1 \lt x \lt 6\)


    • \(-x^2 +5x +6 = 0 \quad\text{se}\quad x = -1 \quad\text{oppure}\quad x = 6\)


    • \(-x^2 +5x +6 \lt 0 \quad\text{se}\quad x \lt -1 \quad\text{oppure}\quad x \gt 6\)


  2. \(7x^4 + 4 \gt 0 \quad\) per ogni \(x \in \mathbb{R}\)


    • \( 6x^2 - x +3 \gt 0 \quad\text{se}\quad x \lt \dfrac{1 -\sqrt{73}}{12} \quad\text{oppure}\quad x \gt \dfrac{1 +\sqrt{73}}{12}\)


    • \(6x^2 - x +3 = 0 \quad\text{se}\quad x = \dfrac{1 -\sqrt{73}}{12} \quad\text{oppure}\quad x = \dfrac{1 +\sqrt{73}}{12}\)


    • \(6x^2 - x +3 \lt 0 \quad\text{se}\quad \dfrac{1 -\sqrt{73}}{12} \lt x \lt \dfrac{1 + \sqrt{73}}{12}\)


    • \(-9x^2 +11x -2 \gt 0 \quad\text{se}\quad \dfrac{2}{9} \lt x \lt 1\)


    • \(-9x^2 +11x -2 = 0 \quad\text{se}\quad x = \dfrac{2}{9} \quad\text{oppure}\quad x = 1\)


    • \(-9x^2 +11x -2 \lt 0 \quad\text{se}\quad x \lt \dfrac{2}{9} \quad\text{oppure}\quad x \gt 1\)


    • \(-4x^2 + 12x -9 \lt 0 \quad \) per ogni \(x \neq \dfrac{3}{2}\)

    • \(-4x^2 + 12x -9 = 0\quad\) se \(x = \dfrac{3}{2}\)
